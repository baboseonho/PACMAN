# Game AI Lab — FSM × Dijkstra

고등학교 2학년 학생을 위한 1차시 인터랙티브 알고리즘 수업 웹사이트입니다. 제공된 `4차시 (3).pdf`의 설명 순서를 그대로 따라가며, 기존 `dijkstra_visual.txt`의 단계 실행·가중치 편집 아이디어를 수업 미션 안에 통합했습니다.

학생은 별도의 난이도 선택 없이 11개 미션을 순서대로 진행합니다. 모든 미션은 Python 코드를 직접 작성하는 과정으로 구성됩니다.

## 실행

Chrome 또는 Microsoft Edge에서 `index.html`을 엽니다. 인터넷 연결, 계정, 설치 과정은 필요하지 않습니다.

브라우저가 로컬 파일 실행을 제한할 경우 `로컬서버_실행.bat`을 실행한 뒤 자동으로 열리는 `http://localhost:8000`을 사용합니다.

## 인터랙티브 실험실

### FSM Lab

거리 슬라이더를 움직이면 몬스터 상태가 즉시 변합니다.

- `distance < 50` → `ATTACK`
- `50 <= distance < 200` → `CHASE`
- `distance >= 200` → `PATROL`

경계값 49·50·199·200에서 상태가 어떻게 달라지는지 직접 확인할 수 있습니다.

### Map Lab

학생이 코드 입력창에 작성한 15 × 15~25 × 25 Python 행렬을 팩맨 스타일의 실제 타일 지도로 변환합니다.

- 행렬 변수 이름은 자유롭게 선택
- `0`: 펠릿이 놓인 이동 경로
- `1`: 파란 팩맨 미로 벽
- `"P"`: Pac-Man 위치, 정확히 한 번 사용
- `"G1"`: 코랄 고스트 위치, 정확히 한 번 사용
- `"G2"`: 파란 고스트 위치, 정확히 한 번 사용
- Map Lab에서 세 캐릭터 좌표와 두 Manhattan distance를 즉시 표시
- 입력 행렬 둘레에 한 칸짜리 자동 벽 생성: N × N 입력 → (N + 2) × (N + 2) 보드
- 예: 4 × 4 입력 → 외곽 벽을 포함한 6 × 6 보드
- 모든 행의 길이와 전체 행 수가 같은 N × N 구조 검사
- 작은 맵에서 시작 즉시 추격당하지 않도록 `"P"`와 각 고스트의 Manhattan distance가 8 이상인지 검사
- `Run map` 실행 후 최종 보드 크기, 경로 수와 자동 외벽을 포함한 벽 수 표시
- 모든 비벽 타일이 Pac-Man과 연결되어 있는지 검사하여 고립된 펠릿·고스트 방지
- `Run map`에 성공한 코드는 Play Lab의 `Student map`으로 저장·자동 선택되며, 이후 행렬을 수정해 다시 실행하면 플레이 맵도 교체
- 선택한 Student map과 최신 검증 맵은 진도와 함께 저장되어 페이지를 다시 열어도 유지
- 고정된 지도 정답 없이 학생마다 다른 맵을 만들고 끝까지 플레이 가능

### Path Lab

세 개의 맵을 제공합니다.

- `EQUAL`: 모든 바닥 비용이 1인 기본 최단 경로
- `WEIGHTED`: 짧은 길에 비싼 타일이 있는 가중치 경로
- `WALL MAZE`: 벽을 피해 상하좌우로 이동하는 미로

기능:

- `Step`: 다익스트라를 한 단계 실행
- `Run/Pause`: 자동 실행 또는 일시정지
- `Reset`: 현재 비용을 유지하고 탐색만 초기화
- `Edit cost`: 바닥을 클릭해 비용을 `1 → 2 → 3 → 5 → 9`로 변경
- 속도 선택: `1×`, `2×`, `4×`

색상은 frontier, visited, current, final path를 구분합니다. 최종 경로가 완성되면 총 비용도 표시됩니다.

### Play Lab

수업 화면 안에서 실제 Pac-Man을 플레이할 수 있습니다. 코드 문제와 게임이 나란히 표시되므로 코드를 제출한 직후 고스트 AI의 변화를 확인할 수 있습니다.

- 이동: 방향키 또는 `W`, `A`, `S`, `D`
- 게임: Play, Pause/Resume, Restart
- 효과음: Sound on/off, 시작·펠릿·FSM 상태 전환·일시정지·충돌·재시작·클리어별 구분
- 맵: Maze A, Maze B, 학생이 작성한 Student map
- 게임 상태: 점수, 생명, 두 고스트의 FSM 상태·거리·실행 중인 함수·경로 비용, 현재 AI 완성도

정답으로 완료한 미션의 검증된 로직이 게임 기능으로 단계적으로 활성화됩니다. `Skip mission`으로 넘긴 미션은 다음 화면만 열어 줄 뿐 AI 기능을 켜지 않으며, 나중에 정답을 제출해야 게임에 연결됩니다.

| 완료 미션 | 게임에 적용되는 변화 |
|---|---|
| FSM | 고스트의 현재 상태를 실시간 표시 |
| Condition order + get_distance() | 학생 코드와 같은 Manhattan 거리 및 50·200 기준으로 PATROL·CHASE·ATTACK 전환 |
| Movement cost | 벽 근처 칸을 비용 2로 계산 |
| Dijkstra + find_path() result | CHASE에서 최저 비용 경로를 만들고 `path[1]`로 정확히 한 칸 이동 |
| Recalculate | 고스트가 타일에 도착할 때마다 움직인 Pac-Man 위치를 기준으로 경로 재계산 |
| Integration | PATROL은 교차로 순찰, CHASE만 Dijkstra 실행, ATTACK은 직접 압박하는 최종 FSM 적용 |

11개 미션을 모두 정답으로 완료하면 Play Lab에 `STUDENT AI LIVE`가 표시됩니다. 이때 두 고스트는 서로 다른 동률 경로 우선순위를 사용하되, 둘 다 학생이 작성한 FSM 조건과 Dijkstra 누적 비용·완화·`path[1]` 이동 규칙을 따릅니다. CHASE 경로선은 코랄 고스트는 실선, 파란 고스트는 파란 점선으로 각각 표시됩니다.

코드 입력창에 커서가 있을 때는 방향키가 게임으로 전달되지 않으므로 학생이 코드 안에서 안전하게 커서를 이동할 수 있습니다.

## 진도 저장

완료 미션, 작성 중인 답, 사용한 힌트와 시도 횟수는 브라우저 `localStorage`에 자동 저장됩니다. 상단의 `↻` 버튼은 전체 수업 진도를 초기화합니다. 테마 선택은 별도 저장되므로 진도를 초기화해도 유지됩니다.

채점기나 페이지 오류로 정답 학생이 막히는 상황을 대비해 각 과제에 `Skip mission` 버튼이 있습니다. 확인창을 거친 뒤 현재 답안을 저장하고 다음 미션을 열며, 왼쪽 목록에는 일반 완료 체크 대신 `SKIP`으로 기록됩니다. 이후 같은 미션에 정답을 제출하면 SKIP 기록은 정상 완료로 자동 교체됩니다. 최종 결과 화면에서도 스킵 수를 확인할 수 있습니다.

## 코드 입력과 채점 피드백

- `if`, `elif`, `def`, `len` 같은 일반 Python 단어와 학생이 같은 답안의 윗부분에서 이미 사용한 이름만 자동완성 후보로 표시됩니다.
- 정답에 필요한 상태명, 조건식이나 완성 코드 줄은 자동완성으로 미리 제공하지 않습니다.
- `↑`·`↓`로 후보를 고르고 `Tab` 또는 `Enter`로 입력합니다. `Esc`로 닫을 수 있습니다.
- 자동완성이 열리지 않았을 때 `Tab`은 공백 4칸을 입력하고, `Ctrl + Enter`는 답안을 채점합니다.
- 자동완성은 선택한 단어만 삽입하며 학생이 작성한 `!=`, `<=`, `<`, `>=` 등의 연산자나 다른 코드를 변경하지 않습니다.
- 수식과 상태 범위도 인쇄용 수학 기호로 바꾸지 않고 실행 가능한 Python ASCII 연산자를 그대로 표시합니다.
- 채점은 문제에서 요구한 비교 연산자를 정확히 검사합니다. 예를 들어 `!=`를 `not in`으로, `<`를 `<=`로 바꾸면 같은 답으로 간주하지 않습니다.
- 긴 답안을 작성하면 코드 입력창이 최대 높이까지 자동으로 늘어납니다.
- 사용한 힌트는 1번부터 3번까지 누적 목록으로 남아 이전 힌트를 다시 볼 수 있습니다.
- 채점 시 탭, 4칸 단위 들여쓰기, 블록 다음 줄의 깊이, 콜론, 괄호·따옴표와 비교 연산자 문법을 먼저 검사합니다.
- 같은 결과를 만드는 학생 정의 변수명은 달라도 인정하지만, Python 문법과 들여쓰기가 틀리면 정답으로 처리하지 않습니다.
- FSM Lab의 `distance_to_player` 슬라이더는 값에 맞춰 고스트와 Pac-Man 사이의 실제 화면 거리와 상태를 함께 바꿉니다.
